# 硬件知识问答

用于回答用户没有要求完整配置单、但在问硬件选择、性能释放、兼容取舍或软硬件协同的问题。典型问题包括“Intel 还是 AMD”“办公要多大内存”“Win10 还是 Win11”“9950X3D 能不能风冷”“电源模式怎么选”“要不要 PCIe 5.0 SSD”“XMP/EXPO 要不要开”。

## 回答边界

- 只回答和台式机硬件选型、性能释放、散热、供电、兼容、BIOS/驱动/系统性能模式相关的问题。
- 不写系统安装教程、激活教程、安全隔离策略、软件使用教程或自动优化脚本。
- 不强行输出整机配置单；用户追问预算或型号时，再回到 `routing.md`、`scenarios.md`、`workflows.md` 查候选或跑兼容。
- 结论先行，默认用 2-4 点解释理由。依赖预算、用途、显示器、已有配件或软件版本时，先给默认假设，再列需要补充的信息。
- 可以用“通常 / 默认 / 更推荐 / 预算紧时”表达取舍，不把社区经验写成绝对规则。

## 高频问答规则

### 先问清楚什么

- 只问硬件选择但没有预算时，先按“主流新机、中文市场、非二手、价格参考日期为当前库 metadata”回答，再提示预算会影响平台取舍。
- 和游戏有关，必须看分辨率、刷新率和游戏类型；只说“打游戏”时默认 2K 3A 和主流网游混合，不直接上旗舰 CPU。
- 和生产力有关，必须看软件、素材分辨率/模型规模、是否多软件并开、是否需要 CUDA/硬件编码/认证显卡。
- 和升级有关，必须看现有 CPU、主板、内存、显卡、电源、机箱、散热和显示器；缺项时先给方向，不编具体兼容结论。

### Intel 还是 AMD

- 普通办公、网页多开、轻剪辑、录屏、直播推流或偶尔转码：默认偏 Intel，理由是核显、硬件编解码、混合核心调度和整机待机体验更省心；新平台建议配 Windows 11。
- 纯游戏 3A：优先把预算给显卡，CPU 够用即可；不要为了更高 CPU 挤掉显卡、电源、SSD 和散热。
- 电竞 FPS、P 社策略、模拟经营、缓存敏感游戏：优先 X3D，尤其是 9800X3D 或低预算 AM4 新品 X3D 例外路线。
- ComfyUI/SD/文生视频、本地 AI 或 CUDA 生态：显卡优先 NVIDIA，CPU 在够用基础上兼顾内存和 SSD。
- 重度多线程、渲染、编译、视频导出：按核心数、编解码、功耗、平台成本和软件生态比较 AMD Ryzen 9 / Intel Core Ultra，不用单一品牌规则覆盖所有场景。
- 用户已有平台或价格差异明显时，先算平台总成本：CPU、主板、内存、散热、电源和后续升级空间一起看。

### CPU 够不够和瓶颈怎么看

- 不使用“瓶颈计算器”给百分比结论。先看用途、分辨率、刷新率和游戏类型：4K 3A 多数更吃显卡，1080p/2K 高刷和 FPS 更吃 CPU、缓存和内存延迟。
- CPU 够用不等于越低越好。高端显卡搭太低端 CPU 会影响 1% low、后台任务和后续升级；i3/低端 i5 搭 RTX 5080/5090 只适合临时过渡。
- 纯游戏预算有限时优先显卡，CPU 到够用档即可；生产力、直播、编译、模拟和多任务场景再提高 CPU 权重。
- 对已有配置，只看“显卡占用没满”不够，还要看帧率目标、CPU 单核/多核占用、显存/内存占用、温度、功耗墙和游戏引擎特性。

### 核显还是独显

- 办公、网页、视频播放、轻修图和低功耗小主机可以用核显，优先 Intel 带核显或 AMD APU/带核显型号；但多屏高刷、3A、AI 绘图、CUDA、3D 和剪辑特效仍需要独显。
- 直播推流、录屏、H.264/H.265 素材和偶尔剪辑时，Intel 核显有硬件编解码价值；即使配 NVIDIA 独显，也可以保留带核显 CPU 作为编码和排障弹性。
- F 后缀 Intel CPU 没有核显，预算能接受且有生产力/推流/排障需求时优先非 F；纯游戏且价格差明显时 F 系可以省预算。

### Win10 还是 Win11

- 2026 年新装机默认建议 Windows 11。Windows 10 Home/Pro 已在 2025-10-14 结束常规支持；除非有 ESU、LTSC、离线专机或明确旧软件/旧硬件兼容需求，不把 Win10 作为新主机默认建议。
- Intel 12 代及更新的 P 核/E 核混合架构，建议 Windows 11，以便系统调度更好地利用混合核心和 Thread Director。
- AMD AM4/AM5 或传统全大核平台，Win10/Win11 的性能差异通常不是装机主要矛盾；但从安全更新、驱动、后续软件兼容和新功能看，新装机仍优先 Win11。
- 用户明确需要老游戏、老采集卡、老专业软件或特定驱动时，可以建议先核对软件和驱动兼容，再决定是否保留 Win10/双系统/离线方案。

### DDR4 还是 DDR5

- AMD AM5 只能 DDR5；Intel 12/13/14 代要看主板版本，DDR4 和 DDR5 不能混插。
- 4000-6500 元预算、Intel 12/13/14 代性价比路线可用 B760 D4 + DDR4 3200/3600，把钱留给显卡、SSD 和电源。
- 新平台、高预算、生产力、后续升级和 Ultra 200S 默认 DDR5；不要为了便宜 DDR4 放弃平台整体收益，除非预算明确卡死。
- 用户已有 DDR4 内存时，升级模式先看能否保留旧内存和主板；新装机则按当前预算和平台寿命重新评估。

### 电源管理和性能模式

- 台式机插电使用、追求游戏/渲染/跑分最大性能：Windows 电源模式可选“最佳性能”或高性能；主板/显卡控制软件也可切到性能模式。
- 日常办公、网页、多开和轻负载：平衡模式通常更合适，性能差距小，噪音和待机功耗更可控。
- 笔记本、迷你主机和小机箱：优先使用厂商性能模式，并确认插电、散热和风扇策略；不要只改 Windows 电源计划就承诺性能提升。
- NVIDIA/AMD/Intel 显卡驱动中的“最高性能”类选项，适合对特定游戏或生产力软件单独开启；全局开启可能增加待机功耗和发热。
- 不默认建议关闭内存完整性、虚拟化安全、杀毒或系统安全功能。若用户明确只做跑分/电竞专机并接受风险，可说明可能有少量性能收益，但安全性下降，需要用户自行决定。

### 内存容量和频率

- 纯办公至少 16GB，更推荐 32GB；网页多开、微信/QQ/浏览器/Office/轻修图同时开，32GB 更稳。
- 主流游戏和新平台默认 32GB；只在极低预算或旧机过渡时用 16GB，并说明后续优先升级。
- 本地 AI 编程/终端 agent、IDE、Docker、多项目、多浏览器标签和本地知识库：32GB 起步，多 agent 并发或容器常驻建议 64GB。
- PS/Lightroom、入门剪辑、轻 CAD：32GB 起步；4K 剪辑、AE、Blender、UE、大型 CAD 或多软件并开建议 64GB，重度项目再看 96GB/128GB。
- DDR5 默认优先两条套装，不默认四条。四条仍是双通道，但更容易增加内存控制器压力，可能降低高频稳定性。
- AMD AM5 日常优先 DDR5-6000 CL30/32；X3D 对内存频率不如非 X3D 敏感，不必为极高频内存牺牲显卡或 SSD。Intel DDR5 平台可按预算和主板能力看 6400-7200+。
- 装好后建议开启 XMP/EXPO，再做稳定性测试；不开 XMP/EXPO 时内存可能只跑 JEDEC 基础频率。

### 两条还是四条内存

- 新装机默认两条套装：32GB 用 16GB×2，48GB 用 24GB×2，64GB 用 32GB×2，96GB 用 48GB×2。
- 四条 DDR5 仍是双通道，不会变四通道；优势是容量和插满外观，代价是更难高频稳定，尤其 AM5 和高频 DDR5。
- 用户明确要插满、已有两条想补两条、或容量需求超过两条可承受范围时，可以用四条，但要提示可能需要降频、更新 BIOS 或手动调参。
- 不建议混不同容量、不同颗粒、不同频率或不同套装；能买同套装就买同套装，不能时按最低共同频率和稳定性预期说明取舍。

### 散热怎么选

- 65W 级 CPU：普通单塔或主流风冷即可。
- 105W、i5/i7 K/KF、R7/R9 非极限负载：强风冷、双塔或 240 水冷按噪音、机箱和预算取舍。
- 150W+、长时间渲染/编译/导出、i9/R9 高端平台或追求低噪：优先 360 水冷或高规格 280/360 水冷，并确认机箱冷排位和风道。
- Ryzen 9 9950X3D 属于 16 核高端 X3D，官方规格为 170W TDP，并推荐液冷以获得更好性能表现。顶级双塔风冷加好风道可以用，但默认更建议 360 水冷；长负载和低噪需求下不要用入门风冷硬压。
- 海景房和展示型机箱预算充足时可用同色 360 水冷，甚至带屏水冷；纯性能和低预算不要为了屏幕/灯效牺牲显卡、电源、SSD 和内存底线。

### 电源怎么选

- 先看 CPU + 显卡功耗，再留 35%-50% 余量；不是瓦数越大越好，电源品质、保护、线材和转换效率同样重要。
- RTX 5080 及以下多数均衡配置，高品质 850W ATX 3.x 通常足够；RTX 5090/5090D、静音余量、后续升级或高端全家桶再考虑 1000W+。
- RTX 40/50 高端卡优先原生 12V-2x6/12VHPWR 线材；没有原生线材不一定是硬错误，合规转接线可用，但白色海景房、好看理线或用户明确不要转接时，优先原生线材和同色电源。
- 普通 ATX/MATX 机箱默认 ATX 电源；SFX/SFX-L 小电源只在 ITX、小型机箱、NAS、卧式或用户明确复用时考虑。

### 主板怎么选

- 不只看芯片组。默认看供电规模、VRM 散热、BIOS 成熟度、内存支持、M.2/USB/网络、版型、售后和价格。
- 低预算办公/入门游戏可以用 H610/A620/H810/A820，但中高端 CPU、K/KF、X3D 或长负载更建议 B760/B860/B650/B850 及以上。
- 视频剪辑、外置高速盘、采集卡、多硬盘和工作站扩展：提高 X870/X870E、Z790/Z890 或同等级高 I/O 主板权重，并复核 M.2、SATA、USB4/雷电和 PCIe 插槽共享。
- 多 M.2、PCIe 5.0 SSD、USB4、采集卡、10G 网卡、声卡等扩展同时存在时，必须看主板说明书的通道共享；普通单 SSD 游戏机不需要为极端扩展能力过度加钱。

### 主板 BIOS 和新 CPU 支持

- 新 CPU 搭旧芯片组主板时，必须确认 BIOS 版本；例如 AM5 新 CPU 搭 B650/X670、Intel 新批次 CPU 搭老批次 B760/Z790，都可能需要更新 BIOS 后点亮。
- 没有旧 CPU 可点亮时，优先选择支持 BIOS Flashback/Q-Flash/M-Flash 等免 CPU 刷 BIOS 的主板，或让商家确认已刷到支持版本。
- 高端 CPU 不只需要“能点亮”，还要看供电和 VRM 散热；弱供电主板配 i7/i9、K/KF、Ryzen 9 或 X3D 长负载，可能跑不满或温度高。
- 13/14 代 Intel 高端平台必须确认 BIOS/微码和 Intel 默认设置；不要用旧 BIOS、无限制功耗和低端散热去压高端 K/KF。

### B 板还是 Z/X 板

- Intel 非超频游戏/办公/直播配置，B760/B860 通常够用；Z790/Z890 主要给 CPU 超频、更多 I/O、多 M.2/雷电/高速扩展和高端供电。
- AMD 游戏和普通生产力，B650/B850 通常足够；X870/X870E 更适合 USB4、多高速盘、采集卡、外置高速盘、旗舰供电和高端展示配置。
- 用户只问“能不能用低端板”，回答要区分能点亮、能稳定满载、是否值得。能点亮不代表推荐。

### SSD 怎么选

- 系统盘默认 TLC NVMe SSD；4000 元附近或办公入门可临时用 512GB，但主流新机建议 1TB 起步，1 万元以上常规建议 2TB。
- 纯游戏加载和日常办公，PCIe 4.0 TLC 已足够；PCIe 5.0 SSD 更适合素材盘、缓存盘、大文件读写、视频剪辑和高端生产力。
- PCIe 5.0 SSD 发热更高，必须确认主板散热片、机箱风道和价格是否合理；不要只因跑分高就牺牲显卡或内存预算。
- 不把机械硬盘作为系统盘；机械盘只适合归档、冷数据或预算明确的大容量仓库盘。

### 单盘还是多盘

- 普通游戏/办公新机默认一块 1TB 或 2TB TLC NVMe，更简单也更省预算。
- 剪辑、AE、达芬奇、Blender、UE、本地 AI 或大量素材：可分系统盘、项目/素材盘、缓存盘，至少保证系统盘和项目盘都是 TLC SSD。
- 多盘配置必须复核主板 M.2 数量、散热片、通道共享和机箱硬盘位；不要只因为主板写了多个 M.2 就忽略显卡通道或 SATA 共享。
- 机械硬盘可做冷数据和备份，不建议承担工程缓存、游戏主库或系统盘。

### 显卡怎么选

- 3A/4K 游戏：显卡优先，CPU 够用即可；不要低预算硬堆高端 CPU。
- FPS 高刷：CPU、缓存和内存延迟会明显影响 1% low，X3D 权重上升。
- ComfyUI/SD/文生视频/本地模型：优先 NVIDIA 和显存容量；显存不足比小幅 CPU 差距更伤体验。
- 直播推流：NVIDIA NVENC 和 Intel 核显编码都有价值；游戏同时直播时，还要给 CPU、内存和散热留余量。
- AMD 显卡适合纯游戏光栅性价比和大显存取舍；Intel Arc 适合预算卡位和视频编解码取舍，但游戏兼容和驱动要提示复核。

### NVIDIA、AMD、Intel 显卡怎么分

- NVIDIA 默认优势在 CUDA、AI、ComfyUI/SD、文生视频、部分生产力软件、NVENC 和光追/DLSS 生态；预算足够且涉及 AI/生产力时优先。
- AMD 默认优势在同价位光栅性能和显存容量；纯游戏、预算敏感、不依赖 CUDA 时可作为强候选。
- Intel Arc 适合预算有限、视频编解码和部分网游/新游戏，但老游戏、驱动兼容和 ReBAR 要提示复核；AI 相关需求不默认选 Arc。
- 显存容量不是唯一标准。还要看芯片档位、显存带宽、功耗、散热、价格和目标分辨率；例如同为 16GB，不代表 5060Ti 16GB 和 5070Ti/5080 是同一档体验。

### PCIe x8、x16 和显卡插槽

- 物理 x16 不等于电气 x16；中端卡若本身是 PCIe 4.0/5.0 x8，在新平台上通常不是问题。
- 老平台 PCIe 3.0 + x8 显卡可能有更明显损失，尤其显存较小、纹理压力大或数据交换频繁的游戏；旧机升级时要提示这个取舍。
- 显卡必须插主板第一条 CPU 直连显卡槽；插到底部芯片组槽可能降速，影响性能。

### BIOS、驱动和基础设置

- 新装机先更新到稳定 BIOS，再装芯片组驱动和显卡驱动；高端 Intel 13/14 代平台尤其要确认 BIOS/微码和默认功耗设置，避免旧 BIOS 异常电压或不稳定。
- 开启 XMP/EXPO 后要做内存稳定性检查；不稳定时先降频或换官方 QVL/成熟套条，不要一味加电压。
- 新平台和独显一般建议开启 Above 4G Decoding / Resizable BAR；Intel Arc 显卡更依赖 ReBAR。若遇到老游戏/老设备兼容问题，再按问题回退。
- HAGS、游戏模式、驱动低延迟模式等设置不要写成固定万能答案：可以按游戏或软件试开，并以实测帧率、1% low 和稳定性为准。

### 机箱和风道怎么选

- 纯性能和无光配置优先风道机箱、网孔前面板、合理进出风和显卡限长余量，不为玻璃展示牺牲温度和噪音。
- 海景房/展示机箱优先水冷和同色系风扇，但要把风扇、冷排、线材和电源颜色计入预算；预算紧时不要为了外观牺牲显卡和电源。
- 养宠、防毛、防尘家庭优先带防尘网、正压风道、易拆洗滤网和风道型机箱；报告里提示定期清理滤网和风扇。
- 显卡长度、厚度和供电线弯折空间都要留余量；机箱限长刚好等于显卡长度，不应写成无风险。

### ITX、小机箱和背插

- ITX 不是单纯换小主板。要同时核对 Mini-ITX 主板、SFX/SFX-L 电源、显卡长度/厚度、散热高度/冷排位、风扇位、硬盘位和线材空间。
- 同预算下 ITX 往往更贵、更热、更难装，适合明确追求体积的用户；默认普通装机不要主动转 ITX。
- 背插/BTF/Project Zero 主要价值是理线和展示，不提升性能。必须同时选背插主板和支持背插开孔的机箱，并确认显卡供电生态、线材和售后。

### 升级顺序怎么判断

- 游戏升级优先看显卡，其次是 SSD、内存、CPU；但如果没有 SSD 或内存低于 16GB，先补基础体验。
- 旧机升级显卡前必须核对电源功率、显卡供电、机箱限长和 CPU/平台是否明显拖后腿。
- 办公卡顿先看是否机械硬盘、内存不足、后台软件和系统盘空间；不要一上来只换 CPU。
- 生产力升级先看内存、SSD 和显卡显存是否不足，再看 CPU 核心数和平台 I/O。

### 主板 Wi-Fi、蓝牙和网络

- 需要无线耳机、手柄、蓝牙键鼠、手机互传或无法布网线时，优先选带 Wi-Fi/蓝牙的主板，后装 PCIe 网卡也可以，但会占插槽和空间。
- 游戏低延迟仍优先有线网；Wi-Fi 6/6E/7 能改善无线带宽和拥塞，但路由器、距离、墙体和干扰同样重要。
- 直播、NAS、素材服务器和局域网拷贝大文件时，可提高 2.5G/10G 网口权重，并同步考虑交换机和 NAS 是否支持。

### 显示器是否纳入配置

- 用户没提显示器时，不默认推荐显示器，也不把显示器计入主机预算。
- 用户说“带显示器”时，通常按整机预算约 10% 预留屏幕；预算很紧时先保主机基本性能。
- 3A 常规：RTX 5070 以下优先 2K，RTX 5070Ti 以上可看 4K；RTX 5070 硬要 4K 可推荐 4K 160Hz，但需说明画质和帧率取舍。
- FPS 常规：优先 1080p/2K 高刷，按 180Hz/240Hz/300Hz+ 目标选择，不默认上 4K。

## 输出示例口径

用户问“办公 Intel 还是 AMD”：

> 办公场景我会优先建议 Intel，尤其是你有网页多开、轻剪辑、录屏或偶尔转码时。原因是 Intel 核显和硬件编解码更省心，Windows 11 下混合核心调度也更适合后台多任务。AMD 不是不能用，如果同价位平台便宜很多或你更看重后续 AM5 升级，也可以选 AMD。

用户问“剪辑和网页多开至少多大内存”：

> 你的场景主要吃内存和缓存。至少 16GB 能用，但我更建议 32GB 起步；如果是 4K 剪辑、AE、达芬奇调色或浏览器/素材/聊天工具长期并开，建议 64GB。

用户问“9950X3D 能不能风冷”：

> 能用顶级双塔风冷，但我默认更建议 360 水冷。9950X3D 是 16 核高端 X3D，官方规格 170W TDP，并推荐液冷以获得更好性能表现。风冷方案要有好机箱风道，并接受长负载下温度、噪音和持续 Boost 可能不如水冷。
