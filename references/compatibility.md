# 兼容性检查

兼容性由 `scripts/check_compatibility.py` 执行，最终配置必须使用 `--strict --require-complete`。不要手动替代脚本判断。

`power_w` 用于电源余量检查，按可见峰值、PPT/MTP 或保守负载估算维护，不等同于厂商页面的默认 TDP；不要据此向用户宣称官方 TDP。

脚本把“是否存在硬不兼容”和“证据字段是否完整”分开报告：`status=complete` 表示未发现硬不兼容且没有待复核项；`needs_review` 表示没有硬错误，但仍有警告、信息或需要复核的跳过项；`incompatible` 表示存在硬错误。为兼容已有调用，单独使用 `--strict` 仍只在硬不兼容时返回非零；加 `--require-complete` 后，完整通过返回 `0`，硬不兼容返回 `1`，待复核返回 `2`。

脚本覆盖 CPU/主板 socket、内存代际/数量/容量、显卡限长、M.2/SATA 数量、电源功率、显卡供电、散热限高/冷排位、机箱主板版型和电源规格；若机箱或电源有 `psu_length_mm` / `psu_max_length_mm` / `psu_clearance_mm` / `length_mm` 字段，还会复核电源限长。`psu_length_recommended_mm` 是保守建议值，超过建议但未超过物理上限时只能写成“尺寸上可能可装，需按冷排、显卡、硬盘笼和线材复核”；`psu_length_condition` 必须原样保留为复核条件。电源本体长度不包含模组插头和线材弯折空间。部分 ITX、卧式、小型机箱会另有电源限高、转接支架和硬盘笼空间限制；库内没有结构化字段时必须列为下单前人工复核项，不写成完整通过。

机箱风扇位属于预算和风道复核项，不是硬兼容项，也不作为 `--strict` 阻塞条件。海景房或好看需求使用水冷时，优先读取机箱 `fan_mounts` / `fan_slots_count`：360 水冷占 3 个 120 风扇位，240/280 水冷占 2 个，420 水冷占 3 个 140 风扇位；剩余风扇位按预算预留。用户明确要求水冷夹汉堡/双面风扇时，冷排自带风扇不抵扣加装风扇预算，但必须提示冷排厚度和空间复核。字段缺失时提示用户自行核实机箱风扇信息，并建议预留加装风扇预算。

如果输出包含 `msg` 或 `skipped`，说明数据字段不足；回复用户时必须保留这些复核点，不能写成“12 类基础检查全部完整通过”。

只有 `--strict --require-complete` 返回 `0` 时，才可写“完整通过”。返回 `2` 时，结论写为“脚本未发现硬不兼容，但仍有下单前复核项”。不适用的跳过项不会阻塞完整度；`memory_freq_max` 只记录主板官网标称的最高超频频率，不等于任意 CPU、四槽布局和任意套条都能稳定达到。6000 C28、7200/8000 等精确需求仍要按主板版本、CPU 内存控制器、DIMM 数量、官网 QVL 和 BIOS 版本复核，不阻塞 DDR 代际、插槽数和容量已经通过的普通整机。显卡限长、散热空间、电源形态等硬兼容字段缺失仍会阻塞。

SSD 的 `dram_cache=true` 只在官方资料明确板载 DRAM 时写入；`dram_cache=false` 可表示官方明确 DRAM-less/HMB。SLC Cache、nCache 或仅有高顺序读写参数不能当作独立 DRAM 证据。剪辑、直播录制、AE 缓存、摄影和频繁读写场景若字段缺失，必须在线核对具体容量和版本，不能凭系列印象承诺带缓存。

电源功率检查只覆盖硬下限。RTX 5080 及以下 850W 高品质 ATX 3.x、RTX 5090/静音/升级余量 1000W+ 等属于选型软规则，仍按 `selection-policy.md` 和最终用途说明取舍。

普通 ATX/MATX 机箱默认选择 ATX 电源。SFX/SFX-L/FLEX/TFX 小电源用于 ITX、小型、NAS、卧式或用户明确复用小电源的场景；装入普通 ATX 机箱即使可用转接支架，也默认按不通过处理，不写成推荐配置。

背插/BTF/Project Zero 不由脚本自动推断。用户明确要求背插时，必须人工确认背插主板与背插兼容机箱，普通机箱不能按背插兼容处理。

主板 M.2 / SATA 数量缺失属于普通装机非阻塞项；单 SSD 整机可列为下单前复核点。用户明确多硬盘、旧 SATA 盘复用、剪辑素材盘、NAS/存储或工作站扩展时，必须补查后再判断。

脚本只检查 M.2/SATA 数量，不自动判断 PCIe 通道共享。多 M.2、PCIe 5.0 SSD、USB4/雷电、采集卡、万兆网卡、声卡或扩展卡场景，必须人工查主板说明书/规格页的 lane sharing：有些插槽会让显卡降到 x8、让 M.2 降到 x2/x4，或禁用 SATA/PCIe 插槽。普通单 SSD 游戏机只列为非阻塞提示，不为此默认升级主板。

显卡的物理 x16 插槽长度和电气 x8 不等于同一概念。RTX5060/RTX5060Ti 等中端卡若官方/规格页为 PCIe x8，在 PCIe 4.0/5.0 平台上不默认视为搭配错误；老 PCIe 3.0 平台、旧主板升级或用户明确追求满血带宽时，再列为复核项。
